# This Unexplained Universe

A content site exploring the world's greatest unsolved mysteries with evidence, analysis, and intellectual curiosity.

**Live:** [thisunexplaineduniverse.com](https://thisunexplaineduniverse.com)

## Tech Stack

- Next.js 15 + React 19
- Tailwind CSS
- Markdown articles with gray-matter
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

## Adding Articles

Create a new `.md` file in `src/content/articles/` with frontmatter:

```yaml
---
title: "Article Title"
description: "Meta description"
date: "YYYY-MM-DD"
category: "Category Name"
image: "/images/articles/filename.jpg"
imageAlt: "Image description"
keywords: ["keyword1", "keyword2"]
---
```
